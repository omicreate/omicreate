# セキュリティ方針

## 対象

この文書は、プロフィールリポジトリ内の公開文書、画像、設定ファイルを対象にします。

## 公開してはいけない情報

- APIキー、トークン、Cookie、認証情報
- 個人端末の絶対パス、作業フォルダ名、端末固有の一時フォルダ
- 非公開の起動URL、検証URL、作業用URL
- 個人名、メールアドレス、住所、電話番号などの個人情報
- 未公開プロジェクト名や内部メモ

## 公開前チェック

公開前に次を実行します。

```bash
npm run check:public-info
```

このチェックは、公開文書やコードに個人環境由来の情報が混ざっていないかを確認するためのものです。

## 基本方針

- 公開プロフィールには、公開されても問題ない情報だけを置く
- 公開前に差分とチェック結果を確認する
